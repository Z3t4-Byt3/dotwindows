TOPIC
	about_FarInventory

SHORT DESCRIPTION
	Computer inventory tools for Far Manager.
	Author: Roman Kuzmin

LONG DESCRIPTION
	This module tools get and show various information about a computer. Some
	types of returned objects are only for viewing but some others allow to
	change their state permanently by their members.

	By default all the commands operate on the current computer but some
	commands allow to specify other computer names as well.

FUNCTIONS
	Import the module and use Get-Help to get more details about the functions.
	The module includes the following functions:

		Open-EnvironmentPanel
		Open-InventoryPanel
		Open-LogicalDiskPanel
		Open-ServicePanel
		Open-StartupCommandPanel

		Get-Uninstall
		Open-UninstallGridView
		Open-UninstallPanel
