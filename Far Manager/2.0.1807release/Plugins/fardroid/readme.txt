
FARdroid v1.0
~~~~~~~~~~~~~~~~~~~~~~~~
   Плагин к файл-менеджеру FAR v2.0 и  выше. Данная  версия  FAR  находится  в
разработке, поэтому могут возникать проблемы с работой плагина.

  Плагин для работы с файловой системой Android OS.

  Для работы требуется:

  - Устройство с Android OS (либо эмулятор)
  - Активная настройка USB Debugging (в устройстве)
  - Установленный Android SDK и в PATH прописан путь к его утилитам
  - Рутованная прошивка для безпроблемной работы

  Тестировалось на HTC Hero.

Установка
~~~~~~~~~
  Создайте  каталог  fardroid  в Far\Plugins и распакуйте  в  него  содержимое
архива. Перезапустите FAR.  Если вы запускаете FAR с ключом "-co",  то сначала
уберите его (ключ) из команды запуска FAR. После запуска ключ можно вернуть на
место.

Использование
~~~~~~~~~~~~~
  Вызовите плагин из списка расширений (F11)

Настройка
~~~~~~~~~
  Из  меню   FAR  выберите  "Параметры -> Параметры внешних модулей",  затем -
FARdroid. Появится окно конфигурирования.
  Более подробно о настройке плагина можно узнать из его справки.  Нажмите F1,
находясь в окне конфигурирования.

Удаление
~~~~~~~~
1 Закрыть все копии FAR.
2 Воспользовавшись Проводником удалить папку, в которую был установлен плагин.
